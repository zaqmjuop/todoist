Class: Component

Component(param)

组件类,一个组件由一个html文件和一个配置文件js组成. html文件提供组件的template和style. js文件提供组件的JavaScript. 可以对组件进行增加，删除，修改，查询的操作，就像操作HTMLElement一样. 组件之间也可以有父子关系，父子间通过事件进行通讯. 组件实例化后会被分配一个componentId用于查询.

Parameters:
Name Type Description
param Object 必须，组件配置对象
Properties
Name Type Description
name String 必须，用于组件的索引
query String | HTMLElement 必须，标记组件插入的位置，若为字符串则作为querySelector参数
url String 一个相对于执行html文件的路径，表示该组件的视图模板的文件，一般作为第一次实例化时调用一次， param.url被调用一次后会缓存视图模板，组件有3个参数可以获取视图模板 调用优先度 param.template > 缓存 > param.url 按优先度从高到低调用，当存在高优先度的参数时，低优先度参数不被调用
template HTMLElement 可选，组件的html结构，当存在时优先作为视图模板
style HTMLStyleElement 可选，组件的style元素，当param.template存在时作为style加入组件，否则无效
data function 必须，组件的初始数据，从方法中返回一个Object作为数据保存到组件的data属性
present Object 可选，没有实际作用，可以作为初始化时传递数据
selectors Object 可选，索引并保存视图中的元素，形式：{name: selector}，保存到组件的elememts属性中， 形式：{name: document.querySelector(selector)}
methods Object 可选，组件的方法集合，键值类型必须为Function，保存到组件的methods属性中，同时自动绑定this到组件上
created function 可选，一个回调函数，组件实例化后插入到页面之前执行
implanted function 可选，一个回调函数，组件实例化后插入到页面之后执行
